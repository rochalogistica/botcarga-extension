BOTCARGA

Politica de Privacidade

Ultima atualizacao: 07 de abril de 2026

1. Introducao

A extensao BotCarga ("nos", "nosso") respeita a privacidade dos seus usuarios. Esta politica explica como tratamos as informacoes quando voce utiliza nossa extensao para Google Chrome.

2. Dados coletados

A extensao BotCarga nao coleta, transmite ou compartilha nenhum dado pessoal com terceiros. Todos os dados inseridos pelo usuario sao armazenados exclusivamente no navegador local do usuario, utilizando a API chrome.storage.local.

Os dados armazenados localmente incluem:

  • Nome, telefone e rota de interesse de motoristas cadastrados pelo usuario
  • Informacoes de cargas cadastradas (origem, destino, tipo, peso, valor, datas e status)

3. Uso dos dados

Os dados armazenados localmente sao utilizados exclusivamente para:

  • Exibir a lista de motoristas e cargas no painel lateral da extensao
  • Gerar textos formatados de cargas para envio via WhatsApp
  • Facilitar a busca e filtragem de informacoes pelo usuario

4. Compartilhamento de dados

A extensao BotCarga nao compartilha, vende ou transfere dados para terceiros, servidores externos ou qualquer outra entidade. Nenhum dado sai do navegador do usuario.

5. Permissoes da extensao

A extensao solicita as seguintes permissoes:

  • storage: Para salvar dados de motoristas e cargas localmente no navegador
  • activeTab: Para injetar o painel lateral na pagina do WhatsApp Web
  • Host permission (web.whatsapp.com): Para funcionar exclusivamente na pagina do WhatsApp Web

6. Seguranca

Todos os dados permanecem no dispositivo do usuario. A extensao nao faz requisicoes a servidores externos e nao utiliza codigo remoto.

7. Exclusao de dados

O usuario pode excluir todos os dados a qualquer momento desinstalando a extensao ou limpando os dados do navegador. Tambem e possivel excluir motoristas e cargas individualmente pela interface da extensao.

8. Alteracoes nesta politica

Podemos atualizar esta politica periodicamente. Quaisquer alteracoes serao refletidas nesta pagina com a data de atualizacao.
